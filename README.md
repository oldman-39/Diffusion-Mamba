# Diff-Mamba
Two-stage Mamba-based Diffusion Model for Image Restoration 

By Lei Liu , Luan Ma , Shuai Wang, Jun Wang and Silas N. Melo 




Requirements

- CUDA 10.1 (or later)
- Python 3.9 (or later)
- Pytorch 1.8.1 (or later)
- Torchvision 0.19
- OpenCV 4.7.0
- tensorboard, skimage, scipy, lmdb, tqdm, yaml, einops, natsort
